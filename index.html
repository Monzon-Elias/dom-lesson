HTML 5 CANVAS, AUDIO AND VIDEO

The Canvas Object is new in HTML5. Canvas is a container for graphics in which, through script, we can draw graphics. In this instance I've created an animated frame to frame movie of a horse galloping.
The Video element is a new HTML 5 tag. It has many properties accessible by DOM. Here I have created a video element with basic controls and autoplay = true.
The Audio element is a new HTML 5 tag. It shares almost all the video tag properties. Here I have added an audio tag with basic controls, with no autoplay.
